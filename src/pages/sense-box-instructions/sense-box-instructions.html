Programming your senseBox MCU

senseBox MCU Programming

 Hey there! Welcome to your senseXperience with senseXperiments! To use this app with your senseBox MCU you need to upload the right program in advance. These slides will guide you through the process of programming your senseBox to take off with senseXperiments

senseBox MCU Programming

 First thing you need in order to edit and upload the code to your senseBox you need to install Arduino IDE from here. Next you need to install some additional libraries, you can find an instruction therefore at our senseBox:edu book

senseBox MCU Programming

 Your next step will be the download of the programm sketch for this game. You can find it at Github, this link will automatically download the sketch to your computer. Open it with your Arduino IDE.

senseBox MCU Programming

 Now add the MQTT server adress, your Wifi network name and your password to the corresponding position. Before you send the code to your senseBox MCU, put the Wifi-Bee on the XBEE Slot 1. If you're done with that you can connect your senseBox MCU via a MicroUSB cable and upload the sketch to it.

senseBox MCU Programming

 If you and your friends use more than one senseBox MCU at a time, you should additionally adjust your channel name in the programm sketch. The default channel name is "accelerometer". Don't forget to adjust the channel name in your app settings as well.
Now, you're ready to experience the power of acceleration! Have fun
